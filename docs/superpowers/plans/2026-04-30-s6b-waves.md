# Sprint 6B — Wave System Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a sequential wave system to GameLoop — typed event emitter (`wave:cleared`/`wave:started`), wave state tracking, wave-aware win condition, `advanceWave()` public API — and a turbo transition animation in GameScreen triggered by the event.

**Architecture:** `GameLoop` gains a minimal internal event emitter (no external dep) and private `waves`/`currentWaveIndex` state. When the last enemy in a wave dies, it emits `'wave:cleared'` and suspends `update()` until `advanceWave()` is called. GameScreen subscribes on mount, plays a 1500ms animation, then calls `advanceWave()`. `GameLoop.status` stays `'playing'` throughout — the transition is purely a GameScreen concern.

**Tech Stack:** TypeScript, Jest, React Native Animated API (GameScreen animation).

**Prerequisite:** PR `feat/s6-enemies` must be merged before starting this plan.

```typescript
private listeners: Map<GameEvent, Array<() => void>> = new Map()

on(event: GameEvent, handler: () => void): void {
  if (!this.listeners.has(event)) this.listeners.set(event, [])
  this.listeners.get(event)!.push(handler)
}

off(event: GameEvent, handler: () => void): void {
  const handlers = this.listeners.get(event)
  if (!handlers) return
  const idx = handlers.indexOf(handler)
  if (idx !== -1) handlers.splice(idx, 1)
}

private emit(event: GameEvent): void {
  const handlers = this.listeners.get(event)
  if (!handlers) return
  for (const h of handlers) h()
}
```

```typescript
private buildEnemies(level: LevelDefinition): Enemy[] {
  const source = (level.waves && level.waves.length > 0)
    ? level.waves[0].entities
    : level.entities
  if (source.length > 0) {
    return source
      .filter(e => e.entityTypeId !== 'fuel-pickup')
      .map(e => ({
        x: e.x,
        y: e.y,
        alive: true,
        typeId: e.entityTypeId,
        hp: (e.properties?.hp as number) ?? 100,
        xpValue: (e.properties?.xpValue as number) ?? 1,
        movementType: ((e.properties?.movementType as string) ?? 'horizontal') as 'horizontal' | 'vertical',
        burstCount: (e.properties?.burstCount as number) ?? 1,
        dropsPickup: ((e.properties?.dropsPickup as string) ?? null) as 'damage' | null,
        speedMultiplier: (e.properties?.speedMultiplier as number) ?? 1.0,
      }))
  }
  // grid fallback (unchanged)
  const count = level.params.numberOfEnemies
  if (count <= 0) return []
  const cols = Math.min(count, 5)
  const rows = Math.ceil(count / cols)
  const gap = 10
  const totalWidth = cols * ENTITY_SIZE + (cols - 1) * gap
  const startX = Math.round((CANVAS_WIDTH - totalWidth) / 2)
  const enemies: Enemy[] = []
  let placed = 0
  for (let row = 0; row < rows && placed < count; row++) {
    for (let col = 0; col < cols && placed < count; col++) {
      enemies.push({
        x: startX + col * (ENTITY_SIZE + gap),
        y: 60 + row * (ENTITY_SIZE + gap),
        alive: true,
        typeId: 'basic-enemy',
        hp: 100,
        xpValue: 1,
        movementType: 'horizontal',
        burstCount: 1,
        dropsPickup: null,
        speedMultiplier: 1.0,
      })
      placed++
    }
  }
  return enemies
}
```

```typescript
private checkWinLose(): void {
  if (this.state.status === 'card_selection') return
  const allDead = this.state.enemies.length > 0 && this.state.enemies.every(e => !e.alive)
  if (allDead) {
    if (this.waves.length > 0 && this.currentWaveIndex < this.waves.length - 1) {
      // More waves remain — pause and signal GameScreen
      this.waitingForWaveAdvance = true
      this.emit('wave:cleared')
      return
    }
    this.state.status = 'won'
    return
  }
  if (this.state.player.hp <= 0) {
    this.state.status = 'lost'
    return
  }
  if (this.state.player.fuel <= 0) {
    this.state.status = 'fuelEmpty'
  }
}
```

```typescript
advanceWave(): void {
  if (!this.waitingForWaveAdvance) return
  this.currentWaveIndex++
  const nextWave = this.waves[this.currentWaveIndex]
  this.state.enemies = nextWave.entities
    .filter(e => e.entityTypeId !== 'fuel-pickup')
    .map(e => ({
      x: e.x,
      y: e.y,
      alive: true,

      speedMultiplier: (e.properties?.speedMultiplier as number) ?? 1.0,
```
